HTML

  • HTML is defined as a Hypertext Markup Language.
  • HMTL gives the content within the page structure.
  • The head element contains information about the webpage.
  • The meta and title elements are nested within the head element.
  • The body element represents the visible content displayed to the user.
  • The elements that are nested inside other elements are called child elements.

CSS

  • CSS is defined as a Cascading Style Sheets.
  • CSS gives the content style.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • The inline styles are applied to the specific HTML tag, using a style attribute with a CSS rule tos tyle a apecific element only.
  • The internal styles are applied to a specific page, and can configure CSS classes and IDs to style multiple elements within the page code.
  • The external styles are the preferred method (in most cases) because they allow to keep all CSS rules in a separate file and make the design changes easier.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or the attribute to which the rules apply.
  • A declaration contains 2 components: the CSS property that needs to be applied and the value of that property (for example: color(property):blue(value)).
  • Two selectors can be combined into a single CSS rule with a comma, which helps to write DRY (Do Not Repeat Yourself) code.
  • text-align is one of the ways to center the a title or a heading on a webpage.
  • The img selector assigns property values to any image element in the markup.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource.
  • Any HTML element assigned to the class, card, will gain these property assignments from the CSS.
  • Based on learned concepts, the following items were created: margin and padding for the webpage, the style for the header and footer (width, height, background color and color of the text), alignment for the headers h1 and h2, style for the previously added image (display, height, width, margin-left and margin-right), style for the lists (padding and font size), style for the text within the boxes (text-align and font size), and style for the .card elements (width, margin, border and box-shadow).

Git

  • git status: the command that checks what branch the developer is currently on.
  • git checkout -b branch-name: the command that creates anew branch and switches to it.
  • git add -A: the command that add all modications in the current working brnach to the staging area.
  • git commit -m "name of the change": the command that saves made changes.
  • git push: the command that pushes the changes to the remote branch.

JavaScript

  • JavaScript is an object-oriented programming (OOP) language that adds interactivity to a website.
  • A variable is a named container that allows us to store data in our code.
  • An asiggment operator "=" assigns tha specific data that is stored in the variable.
  • The variable contains a string that includes that includes a value.
  • The console.log method is used to output a message to the webconsole and used to test the interactivity of the website.
  • Control flow is the order in which a computer executes code in a script.
  • Typically, JavaScript code is executed from top to bottom. However, using conditional statements such as if statements allows to interrupt control flow.
  • A for loop is used to iterate over the array so that the block of code can be iterated over, or repeated, one for each item in the array.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
  • A function is a reusable block of code that contains instructions to perform a specific task.
  • To declare a function, a function keyword is used and then given a unique name followed by parentheses. It is followed by the curly brackets that contain the code for the task to be performed.
  • Once function is declared, it can be called to produce the expected output. In the example used, functions were listTopics() and selectTopics().
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number.
  • In addition to writting a functional script using JavaScript, a consept of the debugging was introduced. The debugigng process allows to identify the issues within the code by constantly checking the code in the Live Environment to ensure that the desired output in produced.
  • Once the bug is identified, the process is established to determine how to fix it.
